# Java Swing Task Manager 

**Java Swing Task Manager** é um projeto 100% feito em Java para aprendizado sobre **Java Swing** e desenvolvimento de interfaces gráficas(brincar um pouco) 
O objetivo principal é criar um gerenciador de tarefas simples, com possibilidades de melhorias futuras

## Funcionalidades
- Adicionar eventos  
- Modificar eventos  
- Deletar eventos  
- Vincular lembretes (ex.: e-mail)
